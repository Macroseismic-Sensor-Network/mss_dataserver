
Usage
=====

The mss_dataserver provides two command line programs. *mss_dataserver*
controls the data collection, event detection and websocket data serving.
*mssds_postprocess* is used to compute the event supplement data in geoJSON
format and the map visualizations, animations and seismogram images.
