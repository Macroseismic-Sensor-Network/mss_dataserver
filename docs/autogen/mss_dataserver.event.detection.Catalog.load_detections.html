load_detections¶

Catalog.load_detections(project, start_time=None, end_time=None, min_detection_length=None)[source]¶

Load detections from the database.

The query can be limited using the allowed keyword arguments.

Parameters

  • start_time (obspy.UTCDateTime) – The begin of the time-span to load.

  • end_time (obspy.UTCDateTime) – The end of the time-span to load.

  • min_detection_length (float) – The minimum length of the detection.
