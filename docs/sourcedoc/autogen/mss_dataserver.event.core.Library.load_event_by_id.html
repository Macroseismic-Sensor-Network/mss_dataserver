load_event_by_id¶

Library.load_event_by_id(project, ev_id=None, public_id=None)[source]¶

Get an event by the database id or the public id.

Parameters

  • project (mss_dataserver.core.project.Project) – The project to use to access the database.

  • ev_id (int) – The unique database id of the event.

  • public_id (str) – The unique public id of the event.

Returns

The event matching the search criteria.

Return type

Event
